# 🌿 Vert & Soin — Site Jardinier Paysagiste

Site vitrine pour un jardinier paysagiste en Île-de-France.

## Structure

```
jardinier-pro/
├── index.html   # Page principale
├── style.css    # Styles (variables CSS, responsive)
├── script.js    # Animations + formulaire
└── README.md
```

## Sections

- **Hero** — Accroche avec animation de feuilles
- **Services** — Entretien, création paysagère, taille
- **Réalisations** — Galerie de projets
- **Témoignage** — Citation client
- **Contact** — Formulaire + coordonnées

## Pour personnaliser

- Remplacer les images placeholder dans la galerie
- Modifier les coordonnées dans `index.html` (section contact)
- Changer les couleurs dans `:root` dans `style.css`

## Lancer localement

```bash
# Avec Python
python3 -m http.server 8000

# Avec Node
npx serve .
```
